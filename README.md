### Using IntelliJ with Github Tutorial
Melih Arda Aydın

### Accepting the Assignment
1.	Open the assignment link shared via the OIS.
2.	Click “Accept this assignment.”
3.	Once accepted, GitHub Classroom will automatically generate your individual private repository.


### Accepting the Organization Invitation
1.	Navigate to https://github.com
2.	Click on your profile icon (top right corner).
3.	If you see a pending organization invitation, click “Accept invitation.”

### Copying the Repository URL
1.	Open your assignment repository page on GitHub.
2.	Click the green “Code” button.
3.	Ensure that the HTTPS option is selected.
4.	Copy the repository URL provided.

### Connecting Your GitHub Account to IntelliJ IDEA
1.	Open IntelliJ IDEA.
2.	Go to:
      •	Preferences (Mac)
      •	Settings (Windows)
3.	In the search bar, type GitHub.
4.	Select Log in via GitHub.
5.	Complete the authentication process in your browser.
6.	Authorize IntelliJ when prompted.
7.	Close the settings window after successful login.

### Cloning the Repository
1.	On the IntelliJ welcome screen, select “Clone Repository” (Get).
2.	Paste the copied repository URL into the URL field.
3.	If authentication is requested:
      •	Select Log in with Token/Key and generate a key if necessary.
4.	When GitHub opens in your browser, approve all required permissions.
5.	If a token/key is generated, copy and paste it into IntelliJ.
6.	Proceed to complete the cloning process.

### Trusting the Project
   After cloning, IntelliJ may display a security prompt:
   “Trust this project?”
   Select Trust Project to proceed.
